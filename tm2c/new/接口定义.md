# 接口定义

## 处理方法

```
# 元方法
visitItem(ast, context)

# 具体的方法
visitAssignment(ast, context)
```

## 上下文

```

push(ref, value)
pop() -> ref, vlaue
store(name)

getTemp(value) -> tempRef
getTempSize() -> tempSize
restoreTempSize()

```
